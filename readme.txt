== Description ==

This plugins stops spam bots by adding some hidden fields into comment form. This fields are filled in with JavaScript, so only real users with browsers that support JavaScript can post comments on your Wordpress blog/website.

== Installation ==

INSTALATION guides for version 0.0.1:

- Before activating plugin make sure the log.txt is writable (chmod a+rw log.txt under linux OS).
